What I've Made

Good Tunes
Good Tunes is a music-discovery app that allows users to save and rate their favorite tracks from a variety of online music blogs. It is a React app with a Node.JS and Mongo backend. This was my final group project for UNC's Coding Bootcamp.

Friend Finder
Friend Finder is a quiz to find new friends. Users take a quiz and are added into a SQL database, then matched by their answers with compatible friends in the database. It is a full-stack app with a Node.JS back end. This was a homework assignment for UNC's Coding Bootcamp

Train Scheduler
An app for keeping track of fake train schedules. A front-end application using a Firebase database. Users can add trains, calculate when the next train will arrive, and remove trains from the database. This was a homework assignment for UNC's Coding Bootcamp

Pitchfork Scraper
An full-stack app that uses NPM packages to web scrape the best new tracks from Pitchfork, and saves them to a mongo database. It has Node.JS backend and is rendered in handlebars. This was a homework assignment for UNC's Coding Bootcamp, and was later expanded into my final group project, Good Tunes.
